PROBLEMAS IDENTIFICADOS
================================================================================

#1  Algunos algoritmos no funcionan con "memory:"
    → Solución: Detectar tipo de algoritmo + usar disk si es necesario

#2  Procesos largos cuelgan la UI
    → Solución: Usar QThread para ejecución en segundo plano

#3  Referencias entre pasos no se resuelven
    → Solución: Cache de resultados + mapeo step → output

#4  Cancelación no detiene algoritmo
    → Solución: ProcessingFeedbackWithCancel

#5  Tipos incompatibles (puntos vs líneas)
    → Solución: Validación pre-ejecución

#6  Canvas parpadea durante ejecución
    → Solución: blockSignals() + refresh al final

#7  Memoria se agota con capas >500MB
    → Solución: Estrategia adaptativa (tamaño → disk)

#8  Sin feedback en algoritmos largos
    → Solución: VerboseFeedback con timer

#9  Fichero JSON de cadena es gigante
    → Solución: Guardar referencias + gzip

#10 Incompatibilidad entre QGIS 3.16/3.28
    → Solución: Validar versión + mapear algoritmos
