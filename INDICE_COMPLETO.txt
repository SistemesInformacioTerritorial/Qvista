EJECUTABLES (Usar estos):
═════════════════════════

1. qgs_viewer_poc.py
   └─ Viewer simple con cadenas de procesos
   └─ "Construir procesos paso a paso"
   └─ Comando: python qgs_viewer_poc.py

2. qgs_viewer_advanced.py ⭐ RECOMENDADO
   └─ Viewer con DOS pestañas:
      • "Cadenas" (construir desde cero)
      • "Modelos" (ejecutar pregrabados) ← NUEVO
   └─ Comando: python qgs_viewer_advanced.py

3. debug_poc.py
   └─ Validar que Processing funciona
   └─ Sin UI, solo logs de validación
   └─ Comando: python debug_poc.py


MÓDULOS (Importados automáticamente):
══════════════════════════════════════

1. processing_chain_poc.py
   └─ ProcessingChainDock: Construir cadenas
   └─ ProcessingStep: Representa un algoritmo
   └─ ProcessingExecutorThread: Ejecución en background
   └─ ParameterWidget: UI dinámica para parámetros

2. process_models_poc.py ⭐ NUEVO
   └─ ProcessModelsDock: Ejecutar modelos pregrabados
   └─ ProcessModelExecutorThread: Ejecución en background
   └─ ModelParameterWidget: UI dinámica por modelo
