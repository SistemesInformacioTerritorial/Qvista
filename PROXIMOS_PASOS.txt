PREGUNTAS FRECUENTES:
═════════════════════

P: ¿Por qué algunos modelos tienen "model_name" y otros "name"?
R: QGIS antigua versión (XML) vs nueva versión (JSON)
   El código ahora busca ambos automáticamente

P: ¿Puedo usar modelos JSON nuevos si los creo?
R: ✓ SÍ, el código soporta ambos formatos

P: ¿Qué pasa si ejecuto un modelo?
R: Se corre en thread separado, no bloquea interfaz
   Ver barra progreso + mensajes

P: ¿Cómo veo si el modelo ejecutó bien?
R: Busca en Logs
   "✓ Modelo completado: [Nombre]"
   O errores si algo falló

P: ¿Puedo modificar los modelos desde aquí?
R: NO, solo ejecutar. Para editar: QGIS Processing Designer
