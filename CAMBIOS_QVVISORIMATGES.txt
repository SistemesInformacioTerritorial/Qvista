3. NUEVO MÉTODO fitImageToWindow() (líneas 230-272)
   ─────────────────────────────────
   ESTE ES EL CORAZÓN DEL CAMBIO
   
   PROPÓSITO:
   Calcular y aplicar el escalado de imágenes manteniendo el ratio de aspecto
   original, sin distorsiones.
   
   ALGORITMO:
   1. Obtiene el tamaño disponible en scrollArea
   2. Calcula tamaño objetivo considerando el factor de zoom (scaleFactor)
   3. Si la imagen cabe en la ventana, usa scaledToWidth() con Qt.SmoothTransformation
   4. Si no cabe, escala para ajustarse manteniendo proporciones
   5. Si después de escalar a ancho todavía es muy alto, escala por altura
   6. Aplica el pixmap escalado sin setScaledContents()
   
   CARACTERÍSTICAS IMPORTANTES:
   - Usa Qt.SmoothTransformation para mejor calidad
   - Evita valores muy pequeños de ventana (usa 800x600 mínimo)
   - No distorsiona la imagen bajo ninguna circunstancia
   - Respeta el factor de zoom (scaleFactor) para zoom in/out
   
   CÓDIGO:
   ```python
   def fitImageToWindow(self):
       """Ajusta la imagen al tamaño disponible manteniendo el ratio de aspecto"""
       if self.pixmap.isNull():
           return
       
       # Obtener tamaño disponible en la ventana
       availableWidth = self.scrollArea.width() - 20
       availableHeight = self.scrollArea.height() - 20
       
       # Si el tamaño es muy pequeño, usar valores por defecto
       if availableWidth < 100 or availableHeight < 100:
           availableWidth = 800
           availableHeight = 600
       
       # Calcular tamaño original de la imagen
       originalSize = self.pixmap.size()
       
       # Calcular tamaño escalado con el factor de zoom y manteniendo ratio
       targetSize = originalSize * self.scaleFactor
       
       # Si es muy pequeño, usar el tamaño disponible
       if targetSize.width() < availableWidth and targetSize.height() < availableHeight:
           # Cabe en la ventana, usar tamaño escalado
           scaledPixmap = self.pixmap.scaledToWidth(
               int(targetSize.width()),
               Qt.SmoothTransformation
           )
       else:
           # Debe ajustarse a la ventana
           scaledPixmap = self.pixmap.scaledToWidth(
               int(availableWidth * self.scaleFactor),
               Qt.SmoothTransformation
           )
           if scaledPixmap.height() > availableHeight * self.scaleFactor:
               scaledPixmap = self.pixmap.scaledToHeight(
                   int(availableHeight * self.scaleFactor),
                   Qt.SmoothTransformation
               )
       
       self.imageLabel.setPixmap(scaledPixmap)
       self.imageLabel.setSizePolicy(QSizePolicy.Ignored, QSizePolicy.Ignored)
       self.imageLabel.adjustSize()
   ```

COMPORTAMIENTO ANTES vs DESPUÉS
================================================================================

ANTES (Con setScaledContents(True)):
───────────────────────────────────
- Las imágenes se estiraban/comprimían para llenar toda la ventana
- Si la imagen era panorámica (ancho), se comprimía en altura
- Si la imagen era vertical, se comprimía en ancho
- Distorsionaba la imagen original perdiendo aspecto natural
- El zoom in/out también causaba distorsión
- Aspecto visual poco profesional

DESPUÉS (Con fitImageToWindow()):
────────────────────────────────────
- Las imágenes se escalan para caber en la ventana
- Se mantiene siempre el ratio original (ancho/alto)
- Aparecen espacios negros cuando hay diferencia de proporciones
- Zoom in/out mantiene proporciones perfectas
- Aspecto visual profesional, similar a visores modernos
- Mejor experiencia de usuario


EJEMPLO DE FUNCIONAMIENTO
================================================================================

Supongamos una imagen de 1200x800 (ratio 3:2) en una ventana de 1024x768:

ANTES (distorsionado):
├─ Imagen se estiraba a 1024x768 (ratio 4:3)
└─ Aspecto final: 1200x800 → 1024x768 = DISTORSIONADA

DESPUÉS (fit correcto):
├─ Calcula máximo ancho: 1024
├─ Escala: 1200x800 escalada a ancho 1024 = 1024x682
├─ Verifica altura: 682 < 768 ✓ (cabe)
└─ Resultado final: 1024x682 con márgenes negros = PERFECTA

Con zoom in (2.0x):
├─ targetSize = 1200*2 x 800*2 = 2400x1600
├─ Escala a ancho 1024*2 = 2048, altura resultante 1365
├─ Como altura > 768*2, escala por altura
├─ Final: ~1536x1536 aproximadamente
└─ Resultado: RATIO MANTENIDO

NOTAS TÉCNICAS
================================================================================

Qt.SmoothTransformation:
- Proporciona mejor calidad de interpolación
- Es más lento que Qt.FastTransformation pero es aceptable
- Hace que el zoom sea más legible

scaledToWidth() vs scaledToHeight():
- scaledToWidth(n) escala la imagen para que el ancho sea n
- El alto se calcula automáticamente manteniendo ratio
- Mismo con scaledToHeight()

QSizePolicy cambios:
- ANTES: Ignored (ignoraba sugerencias de tamaño)
- AHORA: Expanding (se expande para llenar espacio disponible)
- adjustSize() se sigue llamando para mantener proporciones

ScrollArea:
- Sigue siendo útil para imágenes muy grandes con zoom alto
- Ahora se comporta correctamente con el nuevo sistema
