DOCUMENTO 3: PROBLEMAS_SOLUCIONES_PROCESSINGCHAIN.txt (PRÁCTICO)
================================================================================

★ LECTURA RECOMENDADA PARA IMPLEMENTADORES

Contenido (80KB):

  10 problemas específicos:
  
  #1  Algunos algoritmos no retornan bien en memoria
      → 3 soluciones con código
  
  #2  Las capas intermedias se cargan al proyecto automáticamente
      → Estrategia de marcado y limpieza
  
  #3  Las referencias entre pasos no se resuelven correctamente
      → Cache con mapeo paso → resultados
  
  #4  Cancelación de proceso no detiene el algoritmo
      → ProcessingFeedbackWithCancel
  
  #5  Algoritmos que requieren capas vectoriales específicas
      → Validación pre-ejecución de tipos de geometría
  
  #6  Cambios en el canvas durante la ejecución
      → blockSignals() y refresh al final
  
  #7  Memoria se agota con capas grandes
      → Estrategia adaptativa (tamaño → disk)
  
  #8  Algoritmos con tiempo de ejecución impredecible
      → VerboseFeedback con timer de detección de cuelgue
  
  #9  Guardar cadena genera fichero demasiado grande
      → Guardar referencias + compresión gzip
  
  #10 Compatibilidad entre versiones de QGIS
      → Validar versión + mapeo algoritmos renombrados

Para cada problema:
  • SÍNTOMA: Cómo detectarlo
  • CAUSA: Por qué ocurre
  • SOLUCIÓN: Qué hacer
  • CÓDIGO: Implementación lista

Plus:

  • Validaciones a realizar
